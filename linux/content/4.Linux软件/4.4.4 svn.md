

## svn

```bash


## 安装svnserver
yum -y install subversion 
 
## 初始化 
cd project
 mkdir svn
 cd svn
 svnadmin create project_svn
 
## 启动
 svnserve -d -r /root/project/svn
 svnserve -d -r /root/project/svn --listen-port 8086
 svnserve -d -r  /root/project/svn
checkout svn checkout svn://127.0.0.1/ project_svn
```
