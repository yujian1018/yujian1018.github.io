## 收藏夹

```bash

#应用程序
cd /usr/share/applications
vim app.desktop
# [Desktop Entry]
# Version=1.0
# Type=Application
# Name=脑图
# Icon = /home/yj/tool/DesktopNaotu-linux-x64/resources/app/favicon.png
# Exec = "/home/yj/tool/DesktopNaotu-linux-x64/DesktopNaotu"
# Comment=脑图
# Categories=Development;IDE;
# Terminal=false

```
