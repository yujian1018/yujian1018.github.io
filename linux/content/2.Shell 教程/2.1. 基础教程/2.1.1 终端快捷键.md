### 终端快捷键

| 快捷键 | 说明 |
| :--- | :-----------|
| tab | 补全 |
| ctrl+a | 移动游标到行首 |
| ctrl+e | 移动游标到行尾 |
| ctrl+b | 向左移动 |
| ctrl+f | 向右移动 |
| ctrl+k | 删除此处至末尾所有内容 |
| ctrl+u | 删除此处至开始所有内容 |
| ctrl+d | 删除当前字母 |
| ctrl+w | 删除此处到左边的单词 |
| ctrl+y | 粘贴由ctrl+u，ctrl+d，ctrl+w删除的单词 |
| ctrl+l | 相当于clear |
| ctrl+p | 向上显示缓存命令(向上箭头) |
| ctrl+shift+n | 新终端 |
| ctrl+c | 终止 |
| ctrl+r | 搜索你已经输入的命令 |
| Ctrl+d | 键盘输入结束或退出终端 |
| Ctrl+s | 暂停当前程序，暂停后按下任意键恢复运行 |
| Ctrl+z | 将当前程序放到后台运行，恢复到前台为命令fg |
| Ctrl+a | 将光标移至输入行头，相当于Home键 |
| Ctrl+e | 将光标移至输入行末，相当于End键 |
| Ctrl+k | 删除从光标所在位置到行末 |
| Alt+Backspace | 向前删除一个单词 |
| Shift+PgUp | 将终端显示向上滚动 |
| Shift+PgDn | 将终端显示向下滚动 |
