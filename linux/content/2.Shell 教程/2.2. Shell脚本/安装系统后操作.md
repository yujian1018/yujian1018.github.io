### 配置

```bash
ssh 证书登录 关闭密码和 root 登录。SSH 登录必须 KEY ，禁用掉密码登录，禁用 ROOT 远程  

fail2ban + 防火墙 检测状态 开 ssh 端口访问  

数据库除了 bind ，也要用防火墙限制。

所有端口改成非默认  
不是必须的话不开放外网服务端口, 数据库，缓存这些都只开 127 或者局域网 IP  

进程尽量不以 root 启动  
关注自己网站所用的程序的漏洞  
关注自己所用的开源程序的漏洞  
及时更新版本

```
