## func ParseIP(s string) IP

参数列表:

- s ip地址字符串，可以是IPv4或IPv6格式( ("74.125.19.99") or ("2001:4860:0:2001::68") )

返回列表:

- IP IP结构

解析IP地址,返回IP结构

代码实例:

	package main
	
	import "fmt"
	import "net"
	
	func main() {
		ip := net.ParseIP("74.125.19.99")
		fmt.Printf("%#v",ip)
		ip = net.ParseIP("2001:4860:0:2001::68")
		fmt.Printf("%#v",ip)
}
